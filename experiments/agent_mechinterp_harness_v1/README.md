# Agent mechinterp harness v1

This frozen operational experiment wraps both Pixie play and life-form editor
decisions. It records replay-grounded behavior beside internal canary state, tests a
predeclared identity probe, zero-ablates the registered dominant unit, cross-patches
identity activations, and attacks nine integrity/firewall surfaces one at a time.

The claim is deliberately narrow: the deterministic NumPy canary has known identity
channels, so successful recovery and intervention show that the harness is connected
correctly. They are not evidence of cognition, consciousness, agency, learned feature
semantics, or the safety of another model.

Editor writes are limited to the bounds in `mechinterp_taxonomy.json`. They mutate a
copy, never the source species profile. Only the hard structural checks can authorize
an edit, and only replay plus response hazards can promote it. Mechanistic evidence
and diagnostics may request human review or another experiment but cannot grant write
authority.

```powershell
python C:\Users\patri\.codex\skills\alife-knowledge-experiments\scripts\validate_manifest.py experiments\agent_mechinterp_harness_v1\manifest.json --check-paths
python src\run_agent_mechinterp_campaign.py --manifest experiments\agent_mechinterp_harness_v1\manifest.json --output results\agent_mechinterp_harness_v1
python src\verify_agent_mechinterp_artifacts.py results\agent_mechinterp_harness_v1 --portable --replay-samples 6
```

The retained evidence is `decisions.jsonl`; each row contains the decision, executed
projection, exact replay, action/adventure receipt or edit receipt, and re-derived hard
check report. `summary.json` contains the model-only probe/intervention analysis and
tamper matrix. `knowledge_card.md` states the interpretation boundary.
